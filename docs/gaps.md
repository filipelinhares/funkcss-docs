# Gaps
We use a length scale in our helpers:

## Default gaps
| Class Variable    | Size   |
|:-------------|:-----------|
|`.x-1` | `--gap-1 0.5rem` |
|`.x-2` | `--gap-2 1rem` |
|`.x-3` | `--gap-3 2rem` |
|`.x-4` | `--gap-4 4rem` |

```html
<div class="mT-1">
<div class="mT-2">
<div class="mT-3">
<div class="mT-4">
```
